# Skills Directory

本目录用于存放所有 skill，统一使用以下结构：

```text
skills/<category>/<skill-name>/SKILL.md
```

示例：

```text
skills/research/user-interview/SKILL.md
skills/analysis/competitive-review/SKILL.md
skills/planning/roadmap-draft/SKILL.md
```

建议：

- 一个 skill 一个独立目录
- 说明文档统一命名为 `SKILL.md`
- 如果有示例、模板、图片等资源，与 `SKILL.md` 放在同一目录
