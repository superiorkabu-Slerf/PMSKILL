# 需求 / 方向分析结论

该模板用于整理最终输出，不要求逐字照搬。

建议分成两部分：

- 第一部分：判断这个需求或方向是否成立
- 第二部分：沉淀成团队速读摘要

如果当前使用场景更适合对话式、纪要式或表格式输出，可以调整表达方式；但建议尽量覆盖以下内容。
