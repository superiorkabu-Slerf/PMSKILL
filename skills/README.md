# Skills Directory

本目录用于存放所有 skill，按来源与用途分成 3 个主目录：

```text
skills/my-skills/<skill-name>/SKILL.md
skills/product-related-external/<skill-name>/SKILL.md
skills/useful-skills/<skill-name>/SKILL.md
```

示例：

```text
skills/my-skills/product-requirement-analysis/SKILL.md
skills/product-related-external/prd-review/SKILL.md
skills/useful-skills/research-assistant/SKILL.md
```

建议：

- 一个 skill 一个独立目录
- 说明文档统一命名为 `SKILL.md`
- 如果有示例、模板、图片等资源，与 `SKILL.md` 放在同一目录
- `my-skills` 优先存放你自己的核心 skill
- `_template` 作为公共模板目录保留，不属于这 3 个分类中的任何一个
